# 🔒 Security Policy

## Supported Versions

| Version | Supported |
|---------|-----------|
| 1.0.x   | ✅ |
| < 1.0   | ❌ |

## Reporting a Vulnerability

Если вы обнаружили уязвимость:

1. **Не создавайте Issue** — это публично

2. **Отправьте email** на security@example.com:
   - Описание уязвимости
   - Шаги для воспроизведения
   - Версия скрипта
   - Версия ОС

3. **Ожидайте ответа** в течение 48 часов

## GitHub Security

Для серьёзных уязвимостей используйте [GitHub Security Advisories](https://github.com/sergej19882906/x-ui-ultimate/security/advisories).

## Что считается уязвимостью

- ✅ Утечка данных
- ✅ Несанкционированный доступ
- ✅ Обход аутентификации
- ✅ XSS/Injection
- ✅ Проблемы с SSL/TLS

## Не считается уязвимостью

- ❌ Предупреждения антивирусов (false positives)
- ❌ Устаревшие зависимости (без эксплуатации)
- ❌ Проблемы совместимости

## Процесс

1. Получение отчёта
2. Подтверждение
3. Исправление
4. Тестирование
5. Выпуск патча
6. Публикация advisory
